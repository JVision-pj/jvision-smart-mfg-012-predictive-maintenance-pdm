# 預測性維護協作所

透過設備感測數據與演算法預測故障風險，實現從定期保養轉向狀態導向維護。

- 原始系統：預測性維護系統（Predictive Maintenance, PdM）
- 產業分類：智慧製造｜製造管理
- 使用部門：設備工程、廠務、製造工程
- 工作流程：感測數據收集 → 特徵擷取 → 故障預測模型運算 → 風險告警 → 維護排程建議
- AI 應用：機器學習建立設備故障預測模型、AI異常模式辨識早期預警、智慧建議最佳維護時機
- 主要 KPI：非計畫停機率下降幅度、預測準確率、維護成本節省率
- 代表產品：Augury、SKF @ptitude、GE Digital APM、PTC ThingWorx
